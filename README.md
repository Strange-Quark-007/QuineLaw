## Installation 

### Prerequisites

Before you begin, ensure you have the following installed:

- Node.js and npm (Node Package Manager)

```bash
git clone https://github.com/rishabhknowss/QuineLaw.git
```
```bash 
 npm install
 npm run dev
 ```
